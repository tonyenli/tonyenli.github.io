JavaScript 描述語言 JavaScript 描述語言是一種網頁描述語言 (script language)，其前身為 LiveScript，是 NetScape Communications 公司開發出來的網路描述程式語言，不同於由 Sun Microsystems 公司的 Java 程式語言。兩家公司後來共同開發的語言，就叫 JavaScript。
JavaScript 通常建構在 HTML 文件之中，多以javascript 與 HTML 標記區隔；但不是所有的 JavaScript 語言都一定要有 script> /script。由於所有的 JavaScript 程式都是在用戶端電腦執行，當瀏覽器讀取與解譯 HTML 標記語言時，也必須同時讀取與解譯 JavaScript 程式，因此所使用的瀏覽器必須能解譯與執行 JavaScript 程式。這也是為什麼其他網路描述語言 (如 MicroSoft 公司開發的 Jscript 與 VBScript）較不普及的原因。因為 Jscript 或 VBScript 與 NetScape Navigator 瀏覽器不相容，故除了 ASP 動態網頁外，較少使用此類語言。
